What is Blockbot

BLOCKBOT consists of eight types of smart blocks that can form various configurations using our patented electronic assembly blocks and connectors for power supply and communication. Each block contains a 32-bit CPU, allowing it to be connected to a computer and controlled through coding.

It is easy for users to understand and use, and it has the advantage of reusing blocks they already own. By connecting the smart blocks (8 types) with educational/general IoT sensors, it is possible to develop AIoT robot products linked with interactive AI learning (text and image) content.

Coding Block

A coding block that allows programming by pressing four buttons

Light Sensor Block

A block that detects the brightness of light

Battery Block

A block that supplies power to other blocks. It can be charged using a mobile phone charger.

LED Block

A block that can display light in seven different colors.

Bluetooth Block

A block responsible for wireless communication via Bluetooth. It also has a melody function to play music.

Infrared Sensor Block

A block that uses an infrared (IR) sensor to measure the distance to objects.

Motor Block

A block that controls the movement of the Blockbot.

Sound Sensor Block

A block that detects and responds to sound.

How can I assemble the Blockbot ?
